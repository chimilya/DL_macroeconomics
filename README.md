## 1. Описание проекта

Этот проект представляет собой исследование в области макроэкономического прогнозирования с использованием методов глубокого обучения и вариационного вывода. Цель работы — разработать модель, которая на основе набора макроэкономических индикаторов способна:

*   Восстанавливать скрытый (латентный) бизнес-цикл экономики.
*   Строить вероятностный прогноз ключевого макропоказателя (ВВП) на следующий временной шаг.
*   Предоставлять оценку неопределенности прогноза.

В основе решения лежит идея использования **Вариационного Автоэнкодера (VAE)**, который находит низкоразмерное представление ("бизнес-цикл") макроэкономических данных и использует его для генерации прогнозов. Работа включает в себя как этап симуляции данных, так и обучение двух различных архитектур для оценки их эффективности.

## 2. Постановка задачи

**Цель:** Создать модель для макроэкономического анализа и прогнозирования, которая включает в себя:
1.  **Модель распределения данных:** Построить вероятностную модель (VAE), которая описывает совместное распределение наблюдаемых макроэкономических показателей (ВВП, безработица, курс валют, цены на нефть) и скрытого бизнес-цикла.
2.  **Модель генерации данных:** Натренировать модель, способную генерировать новые синтетические временные ряды макропоказателей, статистически похожие на реальные.
3.  **Вариационный вывод:** Применить метод вариационного вывода (через обучение VAE) для аппроксимации апостериорного распределения латентного состояния по данным и для построения вероятностного прогноза.

## 3. Архитектура решения

Проект строится по принципу "Simulation → Modeling → Evaluation".

### Генерация синтетических данных

*   **Латентный процесс (Z):** Моделируется бизнес-цикл как авторегрессионный процесс первого порядка (AR(1)). В него добавлены редкие, но сильные негативные шоки (`shock = -1.5` с вероятностью 0.005), имитирующие экономические кризисы.
*   **Наблюдаемые переменные (X):** Четыре макропоказателя (GDP, U, OIL, RATE) генерируются как линейные и нелинейные функции от латентного состояния Z с добавлением гауссовского шума. Это имитирует связь между реальным "состоянием" экономики и наблюдаемыми индикаторами.
*   **Целевая переменная (Y):** Целевая переменная — ВВП на следующем шаге, сдвинутый на один период вперед относительно входных данных.

### Модель 1: Point-wise VAE (MacroVAE)

Это базовая, "точечная" модель. Она обрабатывает каждый момент времени независимо, как отдельный вектор признаков `X_t`. 

*   **Архитектура:**
    *   **Энкодер:** Многослойная нейронная сеть (MLP) со слоями `Linear` и `LeakyReLU`. Преобразует входной вектор `X_t` в параметры апостериорного распределения латентного состояния: `mu_z` и `logvar_z`.
    *   **Сэмплинг:** Используется трюк репараметризации для получения сэмпла `z` из нормального распределения.
    *   **Декодер:** MLP, который из сэмпла `z` реконструирует входной вектор `X_t_hat`.
    *   **Forecast Head:** MLP, который на основе объединения `z` и `X_t` предсказывает среднее `y_mu` и дисперсию `y_sigma` для ВВП на следующем шаге.

### Модель 2: Sequence-to-Sequence LSTM-VAE (MacroLSTM_VAE)

*   **Архитектура:**
    *   **Энкодер LSTM:** Двухслойный LSTM, который принимает на вход всю временную последовательность `X` (размер `[batch_size, time_steps, features]`) и кодирует её в финальное скрытое состояние `h`. Это `h` используется для вычисления параметров латентного распределения: `mu_z` и `logvar_z`.
    *   **Сэмплинг:** Из полученного распределения семплируется один вектор `z`, который представляет собой *весь* бизнес-цикл для данной последовательности.
    *   **Декодер LSTM:** LSTM, который "разворачивает" этот единый вектор `z` на всю длину последовательности (`time_steps`), выдавая `h` для каждого шага. Затем через `Linear` слой реконструируется вся последовательность `X_hat`.
    *   **Forecast Head:** MLP, который для каждого временного шага, используя соответствующий `z` и `X_t`, предсказывает `y_pred` для следующего шага. На выходе получаем последовательность прогнозов той же длины.

## 4. Основные результаты

Были обучены и сравнены две модели: Point-wise VAE и LSTM-VAE.

**Сравнение эффективности моделей:**

| Метрика           | Point-wise VAE | LSTM-VAE |
| :---------------- | :------------- | :------- |
| **Correlation Z** | 0.9934         | 0.6964   |
| **MSE Z**         | 0.0139         | 0.6073   |
| **Correlation GDP**| 0.9707         | 0.9702   |
| **MSE GDP**       | 0.0585         | 0.0589   |

**Анализ результатов:**

1.  **Качество реконструкции латентного состояния (Z):**
    *   **Point-wise VAE** показывает **почти идеальное** восстановление (`Corr = 0.993`), так как модель работает с данными "точка-в-точку", фактически запоминая взаимосвязь между `X` и `Z`.
    *   **LSTM-VAE** показывает гораздо более скромные результаты (`Corr = 0.696`). Модель обучается на последовательностях и ее латентное представление не обязано полностью совпадать с истинным `Z`, а является его сжатой, "динамической" версией, более полезной для прогнозирования на временных рядах.

2.  **Качество прогноза ВВП:**
    *   Обе модели показывают **близкие результаты** (Correlation ~ 0.97). Это означает, что для прогноза ВВП на один шаг вперед использование сложной LSTM не дает значительного преимущества перед точечной MLP.
    *   С другой стороны, **LSTM-VAE**, несмотря на небольшое снижение в метриках ВВП, является **более правильной и перспективной моделью** для работы с временными рядами, так как она:
        *   Учитывает временную структуру данных.
        *   Может быть расширена для прогнозирования на несколько шагов (Multi-step forecasting).
        *   Дает более релевантное представление бизнес-цикла как динамического процесса.

**Вывод:** LSTM-VAE — это архитектурно более правильное решение для макроэкономических данных, хотя на коротком горизонте прогноза его преимущество не очевидно. Это показывает, что модель может быть улучшена (например, с помощью увеличения объема данных или усложнения симуляции).

## 5. Ключевые улучшения

1.  **Sequence-to-Sequence LSTM-VAE:** Внедрение LSTM в архитектуру VAE позволило модели кодировать и декодировать временные ряды напрямую.
2.  **Проверка размерностей:** Добавлены проверки, гарантирующие, что длины входных, выходных и промежуточных тензоров совпадают, что критически важно для корректной работы LSTM.
3.  **Улучшение стабильности обучения:**
    *   Добавлен `BatchNorm1d` в энкодер для ускорения и стабилизации сходимости.
    *   Применен `LeakyReLU(0.2)` для предотвращения проблемы "умирающих" нейронов.
    *   Введен KL-аннелинг для плавного "включения" регуляризующей компоненты KL-дивергенции.
4.  **Анализ и визуализация:**
    *   Визуализация восстановленного латентного состояния `Z` и вероятностного прогноза ВВП.
    *   Построение графика зависимости будущего ВВП (Y) от восстановленного латентного состояния (z_est) для проверки корреляции.
    *   Проведен анализ чувствительности (Sensitivity Analysis) для изучения влияния `Z` на прогноз ВВП.

## 6. Выводы и дальнейшие шаги

### Текущие результаты
Были успешно построены две рабочие модели VAE для анализа синтетических макроэкономических данных. Обе модели показывают высокое качество прогнозирования ВВП на один шаг (корреляция `> 0.97`). LSTM-VAE, хоть и показала меньшую корреляцию в восстановлении латентной переменной, является более перспективной архитектурой для работы с временными рядами.
